# dynamic-form
## Instructions About Project
> 1: First landing home page that have form what type of input fields and label required you can select.
> 
> 2: You can select and unselect a button is your choice what type of input field need.
> 
> 3: After selecting checkbox you will get a modal form where you have to fill up the required as you want.
> 
> 4: In modal asking how many input fields required(in case of checkbox radio and dropdown asking the how many options).
> 
> 5: After filling number of fields you will get input fields for filling label name what label name you want.
> 
> 6: After selection of all fields click on create-form button.
> 
> 7: After clicking on button you will get new form page what ever you have selected.
