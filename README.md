# dynamic-form
## Instructions About Project
> First landing home page that have form what type of input fields and label required you can select.
> You can select and unselect a button is your choice what type of input field need.
> After selecting checkbox you will get a modal form where you have to fill up the required as you want.
> In modal asking how many input fields required(in case of checkbox radio and dropdown asking the how many options).
> After filling number of fields you will get input fields for filling label name what label name you want.
> After selection of all fields click on create-form button.
> After clicking on button you will get new form page what ever you have selected.
